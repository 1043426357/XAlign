XAlign
======

a useful Xcode plugin to align regular code. it can align xnything as u want.

## Install
```
$ curl github.so/XAlign/build/install.sh | sh
```

## Uninstall
```
$ curl github.so/XAlign/build/uninstall.sh | sh
```

## Usage
### re in ur Xcode
```sh
Xcode -> Edit -> XAlign 
```

### Auto Align Shortcut
```sh
shfit + cmd + x
```
